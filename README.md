# User Management System

A simple User Management System built using **Python, Streamlit, and MySQL**. This project demonstrates CRUD operations with user authentication and input validation.

## Features

- User Signup
- User Login
- Welcome Page
- Update Account
- Delete Account
- Username & Password Validation
- MySQL Database Integration

## Technologies Used

- Python
- Streamlit
- MySQL
- mysql-connector-python

## Project Structure

```
User-Management-System/
│── app.py
│── database.py
│── pages/
│   ├── 1_sign.py
│   ├── 2_login.py
│   ├── 3_welcome.py
│   ├── 4_update.py
│   └── 5_delete.py
│── requirements.txt
│── README.md
```

## How to Run

1. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

2. Run the application:
   ```
   streamlit run app.py
   ```
